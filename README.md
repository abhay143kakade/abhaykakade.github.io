# Abhay Vijay Kakade — Portfolio (GitHub Pages)

This is a simple, responsive portfolio website ready to deploy on **GitHub Pages**.

## Included
- `index.html` — main page
- `style.css` — styles and animations
- `script.js` — typing + reveal JS
- `assets/Abhay_Kakade_Resume.docx` — (if available) copied from uploaded files.
  - Included the uploaded resume (docx) in /assets. Rename to PDF if you have a PDF.

## How to use
1. (Optional) Convert `assets/Abhay_Kakade_Resume.docx` to `Abhay_Kakade_Resume.pdf` and replace the file.
2. Create a GitHub repository named `abhaykakade.github.io`.
3. Upload all files and folders to the repository root.
4. Go to **Settings → Pages** and choose **Deploy from a branch** → select `main` and folder `/ (root)` → Save.
5. Your site will be live at `https://YOUR-USERNAME.github.io` in a minute.

## Customize
- Edit `index.html` to update About, Projects, Contact details.
- Add project cards by copying the `.project-card` block.
- For advanced features use Next.js + Tailwind; this is a simple static starter.

## Notes
- The contact form is a demo only. For real email, use Netlify Forms, Formspree, or implement a serverless function.
